# try-xinput-keybd-rust

XInput対応コントローラーからキーボード入力への変換ツール（最小実装版）

## 必要環境

- Windows 10/11 (x64)
- Rust (MSVC toolchain)
- Xbox互換コントローラー

## ビルド方法

```bash
cargo build --release
```

## 実行方法

### 通常ユーザー権限での実行
```bash
cargo run
```

### 管理者権限での実行
1. 管理者権限でコマンドプロンプトを開く
2. プロジェクトディレクトリに移動
3. `cargo run` を実行

## 操作方法

- XboxコントローラーのAボタンを押す → SPACEキーのKeyDownが送信される
- Aボタンを押し続ける → 何も送信されない（キー押下状態が維持される）
- Aボタンを離す → SPACEキーのKeyUpが送信される
- Ctrl+Cで終了

## 注意事項

- 管理者権限で実行されているアプリケーションには入力が届かない場合があります
- 一部のゲームやセキュリティソフトでは無効化される可能性があります
- あくまで学習・検証用途での使用を想定しています
